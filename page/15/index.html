    • 0
      • мҡҙмҳҒмІҙм ңк°Җ CPUлҘј к°Җм§Җкі  мһҲмқҢ
      • лӘЁл“  мқёмҠӨнҠёлҹӯм…ҳ мҲҳн–ү к°ҖлҠҘ
    • 1
      • мӮ¬мҡ©мһҗ н”„лЎңк·ёлһЁмқҙ CPUлҘј к°Җм§Җкі  мһҲмқҢ
      • н•ңм •лҗң мқёмҠӨн„ұм…ҳл§Ң мҲҳн–ү к°ҖлҠҘ
  • timer
    • нҠ№м • н”„лЎңк·ёлһЁмқҙ CPU лҸ…м җ л§үкё°мң„н•ҙ мқҙкІҢ н•„мҡ”
    • м •н•ҙм§„ мӢңк°„ нқҗлҘё л’Ө мҡҙмҳҒмІҙм ңм—җкІҢ м ңм–ҙк¶Ң л„ҳм–ҙк°ҖлҸ„лЎқ interrupt л°ңмғқмӢңнӮҙ
    • л§Ө нҒҙлҹӯл§ҲлӢӨ 1м”Ё к°җмҶҢ
    • нғҖмқҙлЁё к°’мқҙ 0 мқҙ лҗҳл©ҙ м–ён„°лҹҪнҠё л°ңмғқ
    • time sharing кө¬нҳ„ мң„н•ҙ л„җлҰ¬ мқҙмҡ©
  • device controller
    • IO мһҘм№ҳ м „лӢҙн•ҳлҠ” мһ‘мқҖ CPU
    • м ңм–ҙ м •ліҙлҘј мң„н•ҙ control register, status registerлҘј к°Җм§җ
    • local bufferлҘј к°Җм§җ (мқјмў…мқҳ data register)
    • IOлҠ” мӢӨм ң deivce мҷҖ local buffer мӮ¬мқҙм—җм„ң мқјм–ҙлӮЁ
    • IOк°Җ лҒқлӮҳл©ҙ мқён„°лҹҪнҠёлЎң CPUм—җ к·ё мӮ¬мӢӨмқ„ м•ҢлҰј
    • device driver : OS мҪ”л“ң мӨ‘ к°Ғ мһҘм№ҳлі„ мІҳлҰ¬ лЈЁнӢҙ (мҶҢн”„нҠёмӣЁм–ҙ)
    • divce controller : к°Ғ мһҘм№ҳлҘј нҶөм ңн•ҳлҠ” мһ‘мқҖ CPU (н•ҳл“ңмӣЁм–ҙ)
  • DMA controller
    • м§Ғм ‘ л©”лӘЁлҰ¬м—җ м ‘к·јн•  мҲҳ мһҲлҠ” м»ЁнҠёлЎӨлҹ¬
    • IO мһҘм№ҳк°Җ мқён„°лҹҪнҠё л„Ҳл¬ҙ л§Һмқҙ кұ°лӢҲк№Ң CPUк°Җ л„Ҳл¬ҙ л§Һмқҙ л°©н•ҙ л°ӣм•„м„ң DMA controllerк°Җ н•„мҡ”
    • CPUмқҳ мӨ‘мһ¬ м—Ҷмқҙ deviceмқҳ buffer storageмқҳ лӮҙмҡ©мқ„ л©”лӘЁлҰ¬м—җ block лӢЁмң„лЎң м§Ғм ‘ м „мҶЎ
